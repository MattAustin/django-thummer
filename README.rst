==============
django-thummer
==============
A website snapshot and thumbnailing app for 
`Django <http://www.djangoproject.com/>`_. Uses firefox, selenium, celery, and
sorl-thumbnail.


Getting Started
===============
Documentation is hosted online at: https://django-thummer.readthedocs.io/
