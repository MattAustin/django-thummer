thummer is a website-snapshot and thumbnailing utility. Developed to aid those depending on "free" web thumbnailing services which restrict the number of requests, thumbnail sizes, and use watermarks or require link backs. Set up your own thumbnailing web service! Thummer is a free django web-app which you can install on your own server. Get it for yourself at: https://launchpad.net/thummer.
